User & Server

1. User --> Server
    * login / logoff
    * status

2. Server --> User
    * Online / Offline

3. User --> User
    * direct messages
    * broadcast messages / group messaging

Commands:

    login <user> <password>
    logoff

    msg <target> <body>

    join <#topic>
    leave <#topic>
        #topic <-- chatroom / group chat
